🍃 Green Beverages Financial Report – Power BI Dashboard
📊 Project Overview

The Green Beverages Financial Report Dashboard is an end-to-end Power BI solution designed to provide a comprehensive view of sales performance, financial health, and supply chain insights.

This project focuses on transforming large-scale transactional and sourcing data into interactive, business-ready insights without overwhelming the end user.

🎯 Objectives
Deliver a centralized analytics dashboard for business decision-making
Enable granular time-based analysis (Year → Quarter → Month → Day)
Track financial KPIs and revenue trends
Provide visual storytelling for better understanding of data
🧩 Key Features
🔹 Custom Navigation & UI
Clean and intuitive dashboard layout
Interactive slicer panel:
Year
Quarter
Month
Day
Seamless user experience with easy navigation
🔹 Advanced DAX Calculations

Implemented dynamic measures such as:

Prior Year (PY) Revenue
Year-to-Date (YTD) Performance
Dynamic Net Share %

These measures automatically update based on filters applied to 81K+ generated bills.

🔹 Visual Storytelling

Used advanced visuals to enhance insights:

Sunburst Chart → Customer hierarchy analysis
Treemap → Product category performance
Highlighted top products like Chocolate Protein
Map Visuals → Geographic revenue distribution
🔹 KPI Tracking

Designed high-contrast KPI cards for quick insights:

💰 Net Revenue: $15.5B
📉 Discount Percentage: 5%
